# wagon-kerberos-java
Small modification to fix Kerberos/SPNEGO implementation of Wagon-HTTP-2.9

## Example POM Config
```xml
    <build>
        <extensions>
            <extension>
                <groupId>org.adeptnet</groupId>
                <artifactId>wagon-kerberos</artifactId>
                <version>1.0.1</version>
            </extension>
        </extensions>    
    </build>
```
